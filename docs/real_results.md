# Real-data benchmark results

This report is generated by GitHub Actions from the official UCI **Default of Credit Card Clients** dataset. The dataset contains 30,000 customers and has a default rate of 22.12%.

## Executive result

- Best cross-validated model: **Histogram gradient boosting**
- Best ROC AUC: **0.7861**
- Full-history minus latest-month logistic ROC AUC: **+0.0172**
- Validation: Five-fold stratified customer-level cross-validation; not temporal validation
- Release decision: Research benchmark only; no production promotion without point-in-time UK data

**Decision:** The six-month history has measurable incremental value over a latest-month-only logistic model. A deep sequence model should advance only if it beats the strongest tabular baseline under the same folds and improves calibration or fixed-capacity capture.

## Interpretation limits

The source has no application timestamp, policy decision, interest rate, loss-given-default, treatment assignment, UK geography, or rejected-applicant outcome. Therefore, this benchmark cannot establish temporal robustness, business profit, intervention benefit, UK fairness, or production readiness. Those remain explicit release gates.
