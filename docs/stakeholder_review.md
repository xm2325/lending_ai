# Stakeholder review script

## Recommendation first

State whether the sequence model should remain research-only, move to shadow mode, or proceed to a limited treatment test.

## Evidence

Compare the sequence model with the strongest baseline on an out-of-time cohort. Show the change in ROC AUC, top-capacity recall, exposure capture, calibration and early-warning performance.

## Why the result matters

A small average metric gain is useful only when it identifies additional risky exposure early enough for an approved treatment. Report the affected population, capacity, value assumptions and customer risks.

## Failure analysis

Show where the candidate loses: short histories, missing months, high utilisation, age bands, drifted cohorts, or low-volume groups. Separate sampling noise from a repeatable failure.

## Control plan

Define the champion/challenger period, monitoring thresholds, recalibration route, rollback trigger, model owner and independent reviewer.
